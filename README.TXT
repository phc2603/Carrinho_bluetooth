O carrinho funciona através de conexão bluetooth, via joystick, comando de voz e sensor acelerômetro.
Para isso, basta baixar o aplicativo android feito no app inventor*, e fazer o pareamento de conexão.

OBS: O microcontrolador utilizado foi o ESP-32, por isso há algumas configurações de bluetooth no começo dele.

*Link do aplicativo: 
https://gallery.appinventor.mit.edu/?galleryid=1d370498-7602-4038-bcf7-02f74d56187a
